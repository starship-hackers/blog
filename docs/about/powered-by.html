Powered By

The code that generates this site is hosted on Github. It's written using the following open source languages/libraries:

  • Clojure programming language
  • The Dragon static site generator, which in turn incorporates functionality from:
    • the Stasis library
    • Selmer page templates (Django-based)
    • and more
  • The {less} CSS compiler

The generated content is then published to AWS S3 and served from there.
